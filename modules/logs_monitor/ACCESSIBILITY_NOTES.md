# ACCESSIBILITY NOTES

## Current Coverage
- Page has a labelled main heading.
- Modal uses `role="dialog"` and `aria-modal="true"`.
- Icon-only buttons include accessible labels where needed.
- Tables use header cells and readable text labels.
- Color status is paired with text labels.

## Future Checks
- Keyboard focus trap for modal once modal component is standardized.
- Screen reader verification for live refresh states.
