# UI CONTRACTS

## Application Setup UI Contract
- `/application-setup` must show configurable portal identity and branding controls.
- Screen must include portal title, subtitle, support email, default language, public dashboard toggle, local asset root note, placeholder assets, theme tokens, and unit-level branding.
- Asset manager must show logo, banner, favicon, and public dashboard image placeholders.
- Asset upload controls are placeholders only and must not upload real files.
- Save draft, publish, and reset are visual states only until governed APIs exist.
- Theme token values must stay centralized and readable.
- Unit-level branding must show public dashboard enablement and language preference.
